Here are a couple of demos for the Google Analytics Ez Dashboard library

Here's the official documentation.

Reporting Demos

These demos show you how to add charts / visualizations to your page

  • Set Demo – A simple way to re-use query objects to add multiple charts to a page.
  • Multi-Axis Demo – How to configure multiple axis with line charts.
  • Google Visualization Wrappers Demo – Simple wrapper objects that provide default configurations so the charts look similar to the ones found in Google Analytics.
  • CoreQuery Demo – This object allows you to query the API and not have any chart be associated with the response. Great for developing your own visualizations.
  • Auth Demo – Demonstrates how customize the default authorization look and feel by configuring the onAuthorized and onUnAuthorized handlers.

Control Demos

These demos show you how to add controls to input user data to your page.

  • Query Builder Demo – Demonstrates how to create a simple query builder form for users to input all the values for a Core Reporting API query.
